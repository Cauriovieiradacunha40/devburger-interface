# DevBurger Interface

## Estrutura Principal

A arquitetura principal do frontend foi definida na Aula 03, estabelecendo a organização base do projeto.

A partir deste ponto, a estrutura principal permanecerá praticamente inalterada durante o desenvolvimento. As futuras atualizações deste documento registrarão apenas alterações na árvore de diretórios, como criação de novas pastas, componentes, páginas e demais arquivos da aplicação.
